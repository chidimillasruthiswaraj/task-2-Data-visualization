# Data Visualization and Storytelling

## Project Title

Superstore Sales Data Analysis Dashboard

## Objective

The objective of this project is to create meaningful visualizations from sales data and communicate business insights using charts and dashboards.

## Tool Used

Tableau Public

## Dataset Used

Superstore Sales Dataset (CSV)

## Visualizations Created

1. Total Sales Overview

   * Displays overall sales performance.

2. Sales by Category

   * Shows which product categories generate the highest sales.

3. Sales Trend Over Time

   * Shows changes in sales across different time periods.

4. Sales by Region

   * Compares sales performance across regions.

5. Top 10 Products by Sales

   * Identifies the best-performing products.

## Key Insights

* Technology category contributes significantly to total sales.
* Sales performance varies across different regions.
* Some products generate higher revenue compared to others.
* Sales trends help understand business growth patterns.

## Project Files

* Superstore.csv → Dataset used for analysis
* Superstore_Dashboard.pdf → Final Tableau dashboard export
* README.md → Project description

## Conclusion

This project demonstrates how data visualization helps transform raw sales data into meaningful insights that support better business decisions.
